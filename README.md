## 🐛 수정 이력 (v2.0)

### 수정된 문제
1. **제작예시 버튼 앵커 오작동** → `href="#samples"` 로 통일, `id="samples"` 섹션 추가
2. **영상 샘플 라인 사라짐** → 업종별 6개 카드 그리드 + 필터 버튼 구현
3. **기존 다크 배경** → 완전 화이트 배경으로 리디자인
4. **추상적 카피(Zero/Fast/Safe)** → 구체적 한국어 팩트 카피로 교체
5. **CTA 반복 배치 없음** → 각 주요 섹션 하단에 CTA 추가
